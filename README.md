### What I Build

I design and operate **multi-agent AI systems** — not as a concept, but as production infrastructure running 24/7. My daily work involves LLM orchestration, prompt engineering, tool integration, memory systems, and making autonomous agents reliable enough to trust with real tasks.

**Current focus areas:**

🤖 **AI Agent Infrastructure** — Multi-agent orchestration with persistent memory, RAG pipelines, embedding search, and graph-based knowledge layers. Agents that manage their own state, recover from failures, and coordinate across sessions.

🔌 **Plugin & Extension Development** — Building plugins for AI platforms that solve real operational problems: model rotation strategies, cost optimization, session lifecycle management.

⚡ **Automation & DevOps** — Linux server administration, CI/CD pipelines, cron-based automation, Docker, Proxmox virtualization, firewall configuration, and infrastructure monitoring.

🌐 **Full-Stack Web Development** — TypeScript/JavaScript, Astro, automated content pipelines, SEO optimization, API integrations (WhatsApp, Telegram, Google Workspace, and more).

🎛️ **Creative Tech** — ArtNet-based show control for live events, BPM-synchronized lighting, audio-reactive systems. Where software meets the real world.
